# Sinatra Rest Skeleton

A boilerplate for **Sinatra** Restful-API applications using Rails structure.

## Dependencies

- `Ruby` 2.3+ with default gems: `json`, `rake`, `bundler`
- `PostgreSQL` 9.4+

## What is inside?

- Sinatra 2.0.x
- Sinatra Contrib 2.0.x

- Active Record 5.2.x
- Postgres 0.x

- RSpec 3.x
- Rack Test 0.x

- Pry 0.x
